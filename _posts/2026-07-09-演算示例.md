> 本框架中使用的高斯分布，不视为独立的知觉实体。它仅作为关系契合度的可微度量嵌入系统；其参数 $\mu$ 与 $\sigma^2$ 实时随交换关系更新，因而其作为‘实体’的稳定性完全由关系网络的动态维持。在本工作的计算中，高斯分布始终作为关系的一阶近似语法，而非其本体论前提。

## 1.1 系统设定

* 3个代理：A、B、C
* 2类硬币：1元（面值 $v_1=1.0$）和 5角（面值 $v_2=0.5$）
* 被建模的换算关系：$p(v_2 \mid v_1)$ —— 当代理收到1枚1元硬币时，他预期应获得多少枚5角硬币。

真实环境汇率 $q = 2.0$（即1元 = 2个5角）。代理们不知道这个值，只能通过交换经验来推测。

每个代理的初始信念 $p_i$：

| 代理 | 初始信念 $p_i$ | 含义 |
|:---|:---|:---|
| A | 1.5 | 认为1元只值1.5个5角（低估） |
| B | 2.0 | 恰好知道真实汇率 |
| C | 1.0 | 认为1元只值1个5角（严重低估） |

自由能计算：

$$F_i = (q - p_i)^2 + 0.1$$

其中 $(q - p_i)^2$ 为预测误差，$0.1$ 为固定复杂度代价。

系统总自由能：

$$\Phi = \sum_{i=1}^{3} F_i$$

临界判定阈值：

* $\Phi_{max} = 1.0$（超过此值，系统处于高应力状态）
* $T_{crit} = 2$（连续2步高应力且无法通过微调降低，触发临界判定）

> 本文使用的 $F_i = (q-p_i)^2 + 0.1$ 是完整变分自由能 $$F_i = \mathbb{E}_{q_i}[-\log p(y \mid \theta_i)] + D_{KL}[q_i(\theta_i) \| p_i(\theta_i)]$$ 的一种**简化形式**。该简化仅在假设似然为高斯分布（固定方差）、采用点估计（忽略不确定性）并固定复杂度代价时成立。它旨在直观呈现“预测误差”与“复杂性”此消彼长的核心动力，其数值轨迹服务于机制演示，不替代严格变分贝叶斯计算。

## 1.2 分步演算

### 第0步：初始状态

| 代理 | $p_i$ | 预测误差 $(q-p_i)^2$ | $F_i$ |
|:---|:---|:---|:---|
| A | 1.5 | $(2.0-1.5)^2 = 0.25$ | 0.35 |
| B | 2.0 | $(2.0-2.0)^2 = 0.00$ | 0.10 |
| C | 1.0 | $(2.0-1.0)^2 = 1.00$ | 1.10 |

$$\Phi_0 = 0.35 + 0.10 + 1.10 = 1.55$$

$\Phi_0 = 1.55 > 1.0$，系统初始即处于高应力状态。但尚未发生交换，系统尚未开始微调，因此先不触发临界判定。

### 第1步：A 与 B 完成一次交换

A 用1元向 B 交换，B 给 A 2个5角（真实汇率）。

A 获得了一次符合真实汇率的成功交易经验。他更新模型：$p_A$ 从 1.5 调整为 1.8（向上修正，向真实值靠近）。

B 的交易经验完全符合其模型，无需调整：$p_B = 2.0$。

C 未参与本次交换，不更新。

更新后：

| 代理 | $p_i$ | 预测误差 | $F_i$ |
|:---|:---|:---|:---|
| A | 1.8 | $(2.0-1.8)^2 = 0.04$ | 0.14 |
| B | 2.0 | 0.00 | 0.10 |
| C | 1.0 | 1.00 | 1.10 |

$$\Phi_1 = 0.14 + 0.10 + 1.10 = 1.34$$

$\Phi_1 = 1.34 > 1.0$，仍然高应力，但比第0步有所下降（1.55 → 1.34）。微调在起作用。

### 第2步：A 与 C 完成一次交换

A 用1元向 C 交换，C 给 A 1个5角（C认为1元只值1个5角）。

A 本次收到的5角数量低于他更新后的预期（1.8）。他再次更新模型：从 1.8 下调为 1.6。

C 的交易经验符合其模型：$p_C = 1.0$，无需调整。

B 未参与本次交换。

更新后：

| 代理 | $p_i$ | 预测误差 | $F_i$ |
|:---|:---|:---|:---|
| A | 1.6 | $(2.0-1.6)^2 = 0.16$ | 0.26 |
| B | 2.0 | 0.00 | 0.10 |
| C | 1.0 | 1.00 | 1.10 |

$$\Phi_2 = 0.26 + 0.10 + 1.10 = 1.46$$

$\Phi_2 = 1.46 > 1.0$。A 被拉扯在两个矛盾的真实之间：B 告诉他“2个5角才算公平”，C 告诉他“1个5角就够了”。系统总自由能 不降反升（1.34 → 1.46）。

此时检查临界条件：

* $\Phi_2 > \Phi_{max}$ ✅
* 第1步到第2步，$\Phi$ 未能通过微调降低（1.34 → 1.46），且 A 的更新方向（1.8 → 1.6）正在远离真实值 2.0。但仍需再观察一步。

### 第3步：A 再次与 B 交换

A 用1元向 B 交换，B 给 A 2个5角。

A 再次获得真实汇率经验。但他刚刚下调到 1.6，现在又遇到 2.0。他再次更新模型：从 1.6 调整为 1.8。

B 保持 2.0。

C 未参与。

更新后：

| 代理 | $p_i$ | 预测误差 | $F_i$ |
|:---|:---|:---|:---|
| A | 1.8 | 0.04 | 0.14 |
| B | 2.0 | 0.00 | 0.10 |
| C | 1.0 | 1.00 | 1.10 |

$$\Phi_3 = 0.14 + 0.10 + 1.10 = 1.34$$

回到第1步的状态。系统在 1.34 和 1.46 之间振荡，无法收敛到低位。

临界判定：

* $\Phi_3 > \Phi_{max}$ ✅
* 从第1步到第3步（连续3步），$\Phi$ 未能在连续步骤中低于 1.0，且 A 的模型在 1.6 和 1.8 之间来回漂移，$\partial \Phi / \partial t \approx 0$（既无持续下降趋势）
* 已经持续 $T = 3 > T_{crit} = 2$

$$\text{Crit}(\Phi_3) = 1$$

系统判定：“继续微调将导致僵化（永远在两种估值之间来回摇摆，交换效率持续低下），必须切换模型结构。”

### 第4步：涌现——模型空间搜索

$\text{Crit}=1$ 触发后，系统进入模型空间 $\Omega$ 搜索。

当前可用的候选换算规则：

| 候选规则 | 描述 | 代理支持情况 |
|:---|:---|:---|
| $\mathcal{R}_1: v_1 = 2 \times v_2$ | 1元 = 2个5角 | B支持，A经历过，C反对 |
| $\mathcal{R}_2: v_1 = 1 \times v_2$ | 1元 = 1个5角 | C支持，A经历过，B反对 |
| $\mathcal{R}_3: v_1 = 1.5 \times v_2$ | 1元 = 1.5个5角 | A的中庸折中，B和C都不满意 |

系统评估每个候选规则下，如果强制推行该规则，预期总自由能 $\Phi$ 会是多少：

* 如果推行 $\mathcal{R}_1$：
  + B：误差 0，$F=0.10$
  + A：$p=2.0$，误差 0，$F=0.10$（A 的历史经验支持该规则）
  + C：误差 $(2.0-1.0)^2=1.00$，$F=1.10$（C 需调整，短期内代价高）
  + 预期 $\Phi \approx 1.30$，但 C 有学习空间（适应后可降至低位）

* 如果推行 $\mathcal{R}_2$：
  + C：误差 0，$F=0.10$
  + A：误差 $(1.0-1.8)^2=0.64$，$F=0.74$
  + B：误差 $(1.0-2.0)^2=1.00$，$F=1.10$
  + 预期 $\Phi \approx 1.94$，且 B 将长期处于高误差状态

系统选择使总自由能更低、且能够持续下降的规则：

$$\mathcal{R}^* = \mathcal{R}_1 \quad (v_1 = 2 \times v_2)$$

这个选择不是“投票”，而是基于系统整体生存效率的筛选。

### 第5步：新结构的稳定化与验证

系统强制推行 $\mathcal{R}^*: v_1 = 2 \times v_2$。

A 从 1.8 迅速调整为 2.0（历史经验支持此规则）。

B 保持 2.0。

C 面临选择：要么将 1.0 调整为 2.0（适应新规则），要么退出该系统。

假设 C 调整至 2.0。更新后：

| 代理 | $p_i$ | 预测误差 | $F_i$ |
|:---|:---|:---|:---|
| A | 2.0 | 0.00 | 0.10 |
| B | 2.0 | 0.00 | 0.10 |
| C | 2.0 | 0.00 | 0.10 |

$$\Phi_5 = 0.10 + 0.10 + 0.10 = 0.30$$

$\Phi_5 = 0.30 < \Phi_{max} = 1.0$，系统恢复健康。

模型证据比较：
* 旧模型 $$\mathcal{R}_{old}$$（代理各自为政）：无法解释第1步和第2步的连续冲突，对数证据 $$\log p(\text{历史} \mid \mathcal{R}_{old})$$ 极低（对应 $\Phi$ 持续在 1.34-1.55 之间高位振荡）。
* 新模型 $\mathcal{R}^*$：完美解释第1步和第3步的成功交易（A与B的两次 2:1 交换），并将第2步的“冲突”重新解释为“C 的临时偏离”。

$$\log p(\text{交换历史} \mid \mathcal{R}^*) \gg \log p(\text{交换历史} \mid \mathcal{R}_{old})$$

新结构的有效性通过两项指标得到确认：自由能恢复至低位（0.30 < 1.0），且模型证据显著跃升。

## 1.3 完整数据轨迹表

| 步数 | 事件 | $p_A$ | $p_B$ | $p_C$ | $\Phi$ | 状态 |
|:---|:---|:---|:---|:---|:---|:---|
| 0 | 初始 | 1.5 | 2.0 | 1.0 | 1.55 | 高应力 |
| 1 | A←B（2个5角） | 1.8 | 2.0 | 1.0 | 1.34 | 微调下降 |
| 2 | A←C（1个5角） | 1.6 | 2.0 | 1.0 | 1.46 | 微调失效，$\Phi$ 反弹 |
| 3 | A←B（2个5角） | 1.8 | 2.0 | 1.0 | 1.34 | 振荡，$\partial \Phi / \partial t \approx 0$ |
| 临界判定 | $\Phi > 1.0$ 持续3步 | — | — | — | — | $\text{Crit}=1$ |
| 4 | 涌现搜索 | — | — | — | — | 选择 $\mathcal{R}^*: v_1 = 2v_2$ |
| 5 | 新结构稳定 | 2.0 | 2.0 | 2.0 | 0.30 | 健康，$\Phi$ 降至低位 |

## 1.4 这个演算说明了什么

1. 自由能是可追踪的：每一帧的 $\Phi$ 都可以精确计算（从 1.55 → 1.34 → 1.46 → 1.34 → 0.30）。

2. 临界点有明确的判定条件：$\Phi > \Phi_{max}$ 且微调失效（振荡而非下降），$\text{Crit}=1$ 在连续3步高应力且无下降趋势后被触发。

3. 涌现的规则选择是有依据的：系统在候选规则 $\mathcal{R}_1$、$\mathcal{R}_2$、$\mathcal{R}_3$ 中选择使预期 $\Phi$ 最低且能持续下降的那一个（$\mathcal{R}^* = \mathcal{R}_1$）。

4. 新规则的稳定性可通过两个指标验证：
   - $\Phi$ 降至低位并维持（0.30 < 1.0）
   - 模型证据显著跃升（新规则解释力更强）

5. 概率的角色：新规则稳定后，系统不再要求每笔交换“绝对 2:1”，而是形成一个“可接受范围”（如 1.8-2.2），超出此范围的交换会被拒绝或重新协商——这是新结构中概率作为筛选规则的具体运作。
